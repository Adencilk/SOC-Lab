# 🛡️ SOC Lab Project

## Overview

This project documents my hands-on experience building a Security Operations Center (SOC) lab using Wazuh.

The lab is designed to simulate a real-world SOC environment for monitoring, detecting and analysing security events from different systems.

## Lab Architecture

Host machine:
- Windows laptop
- VirtualBox used for virtualization

Virtual machines used:
- Wazuh server (Ubuntu)
- Ubuntu monitored endpoint (agent)
- Windows monitored endpoint (agent)
- Separate Linux machine used as an attack and testing machine

## Main Components

- Wazuh Manager
- Wazuh Indexer
- Wazuh Dashboard
- Wazuh agents on Ubuntu and Windows systems

## What I implemented

- Installed and configured Wazuh server components
- Installed Wazuh agents on Linux and Windows
- Practiced log collection and alert monitoring
- Performed troubleshooting of services and system resources
- Worked with Linux services, networking and package management
- Designed a multi-machine SOC lab environment

## Skills gained
